Install dependencies by using : 
```
npm install
```

<h1>Start a local server</h1>
To start a server type : 

```
npm run serve
```

If windows you need to first download componenets:


```
bower install
```

Use babel : 
```
babel -w scripts -s --retain-lines -d lib
```

and then : 

```
firebase serve
```
